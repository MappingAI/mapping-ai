## Enrichment Pipeline

1. **Seed** — Add basic info (name, title, org, category)
   ```bash
   node scripts/seed-time100.js  # or create new seed script
   ```

2. **Enrich** — Fill beliefs/stances via Exa + Claude (~$0.04/person)
   ```bash
   node scripts/enrich-people.js --ids=X,Y,Z
   ```

3. **Review** — Check in admin.html, fix any errors

4. **Export** — Push to map
   ```bash
   node scripts/export-map-data.js
   aws s3 cp map-data.json s3://mapping-ai-website-561047280976/
   aws cloudfront create-invalidation --distribution-id E34ZXLC7CZX7XT --paths "/map-data.json"
   ```
